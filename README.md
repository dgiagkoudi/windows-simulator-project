# Windows Simulator
Προσομοιωτής λειτουργικού συστήματος βασισμένος στα Windows 98, που διαθέτει δυναμική διαχείριση παραθύρων, πλήρως προσβάσιμο Start Menu και αυτοματοποιημένο σύστημα παραγωγής alerts.

## Περιγραφή
Η εφαρμογή παρέχει:

- Retro Desktop UI: Πλήρης προσομοίωση επιφάνειας εργασίας με εικονίδια, Taskbar, Start Menu και λειτουργικό ρολόι.

- Window Management System: Δυνατότητα ανοίγματος πολλαπλών παραθύρων με Drag-and-Drop λειτουργικότητα και δυναμικό Z-index.

- Chaos Engine: Σύστημα ιών και σφαλμάτων που αναπαράγει παράθυρα αυτόματα, δημιουργώντας μια διαδραστική εμπειρία "System Overload".

- Advanced Accessibility: Πλήρης πλοήγηση μέσω πληκτρολογίου (Tab, Enter, Escape), χρήση ARIA labels για αναγνώστες οθόνης και υποστήριξη μειωμένης κίνησης.

- Mobile Responsiveness: Πλήρης προσαρμογή του layout και υποστήριξη Touch Events.
